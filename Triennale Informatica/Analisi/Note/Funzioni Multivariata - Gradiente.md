# Funzioni Multivariata - Gradiente
Il [[Funzioni Multivariata - Gradiente|gradiente]] di una funzione reale di più variabili raccoglie tutte le derivate parziali prime.
- Per $f:\mathbb{R}^n\to\mathbb{R}$ si definisce:
$$\nabla f(x)=\left(\frac{\partial f}{\partial x_1},\dots,\frac{\partial f}{\partial x_n}\right)$$
- Il gradiente indica la direzione di massima crescita locale della funzione.
- Un punto critico soddisfa $\nabla f(x)=0$.
- Il gradiente è centrale nello studio di [[Matrice Hessiana|matrice Hessiana]] e [[Moltiplicatori di Lagrange|moltiplicatori di Lagrange]].
