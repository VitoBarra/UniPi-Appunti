# Funzione di attivazione - ELU (Exponential Linear Unit)
---
una [[funzioni di attivazione|Funzione di attivazione]]

$$
f(\alpha, x) =
\begin{cases} 
\alpha (e^x - 1) & \text{se } x \leq 0 \\
x & \text{se } x > 0
\end{cases}
$$
