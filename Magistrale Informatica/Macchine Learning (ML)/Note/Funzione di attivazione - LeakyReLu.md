# Funzione di attivazione - LeakyReLu
---
La [[funzioni di attivazione|funzione di attivazione]] di attivazione LeakyReLu è definita come:
$$
f(x) =
\begin{cases} 
\alpha x & \text{se } x < 0 \\
x & \text{se } x \geq 0
\end{cases}
$$
